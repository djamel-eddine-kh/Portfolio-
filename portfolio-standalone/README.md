# Djamel Eddine Khelifaoui — Portfolio

A dark-theme one-page portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Requirements

- Node.js >= 18
- npm, pnpm, or yarn

## Getting started

```bash
# Install dependencies
npm install
# or
pnpm install

# Start the dev server (opens at http://localhost:5173)
npm run dev

# Build for production
npm run build

# Preview the production build locally
npm run preview
```

## Customisation

All content is static and lives in the section components under `src/components/sections/`:

| File | Section |
|------|---------|
| `Hero.tsx` | Hero / intro |
| `About.tsx` | About me |
| `Experience.tsx` | Work experience timeline |
| `Research.tsx` | PhD research |
| `Projects.tsx` | Project cards |
| `Skills.tsx` | Skill badges |
| `Education.tsx` | Education timeline |
| `Achievements.tsx` | Achievements |
| `Contact.tsx` | Contact links |
| `Footer.tsx` | Footer |

Update the contact links (email, LinkedIn, GitHub) in `src/components/sections/Contact.tsx`.
